# 在线学习论坛

在线学习论坛是一个功能基本全面的内部交流使用论坛平台，提供发布问题、个人资料展示、评论、回复、搜索、接收消息通知等基本功能。后端采用了 SpringBoot + MyBatis + MySQL进行构建，前端采用了 Thymeleaf + Layui 进行展示，界面风格简洁。
